CCG to CSU Data Load

To obtain the input file  

Login into Live TdB as an admin user.
1. Go to Tree Interface
2. Choose “IT Service Provider” hierarchy
3. Select “NHS England”
4. Choose Reports → “Organisation Downloads”
5. On the new screen, select:
            Organisations: Clinical Commissioning Groups
            Type: Organisation Details
6. Press “Run Download”
